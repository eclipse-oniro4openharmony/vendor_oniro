# Community Code of Conduct

This project is governed by the
[Eclipse Community Code of Conduct](https://www.eclipse.org/org/documents/community-code-of-conduct/).

By participating in this project, you agree to uphold this code of conduct.

Instances of abusive, harassing, or otherwise unacceptable behavior may be
reported by contacting the Eclipse Foundation Code of Conduct Committee at
conduct@eclipse-foundation.org. All reports are handled according to the
process described in the Code of Conduct, and the committee is obligated to
maintain confidentiality with regard to the reporter of an incident.
